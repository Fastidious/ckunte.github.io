About this site, etc.

Hello. I am Chetan Kunte, an offshore structures engineer, and producer of this little website — my sandbox for code and prose. I am not on social media; any match there must be a namesake. (Exceptions: GitHub and LinkedIn.)

This domain exists mainly to be a pointer to MX records. I put the idling A record to use, mostly as an afterthought, which turned-on the lights here. Built with Chisel, this site respects web standards, produces valid HTML, CSS, and RSS mark-ups, and is friendly to phones and small screens. For accessibility, handy keyboard shortcuts are available. On MacOS, hold-down Ctrl Opt and press Key. On Linux and Windows, hold-down Alt and press Key:

Syndication. RSS feed channel is available to keep up to speed with the latest published on this site.

Code. To ensure it works as intended, all code published on this site is pre-tested without exceptions. Some of the older code snippets may contain legacy python code. These should still work with last of python version 2. (Most personal projects are in repositories and in gists.)

Privacy. This site does not track visitors. There are no forms seeking visitor inputs, and there are no advertisements, no cookies, and no trackers served from this website.

TLS. Transport layer security on this site is available as an option, i.e., the site is accessible via both https as well as http protocols, and it is not redirected from http to https. This is by intent — to enable low bandwidth users to make best use of cache and help with speed-loading.1
